═══════════════════════════════════════════════════════════════
   GEMINI API KEY SETUP - STEP BY STEP
═══════════════════════════════════════════════════════════════

STEP 1: GET YOUR API KEY
───────────────────────────────────────────────────────────────
1. Go to: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key" or "Get API Key"
4. Copy your API key (looks like: AIzaSyDXXXXXXXXXXXXXXXXX)


STEP 2: INSTALL THE PACKAGE
───────────────────────────────────────────────────────────────
Run these commands:
cd backend
source venv/bin/activate
pip install google-generativeai>=0.3.0


STEP 3: CREATE .env FILE
───────────────────────────────────────────────────────────────
Create a file named: backend/.env

Location: /Users/adityasinghal/Developer/Horcrux/backend/.env

Content (replace with your actual API key):
───────────────────────────────────────────────────────────────
GEMINI_API_KEY=your_actual_api_key_here
───────────────────────────────────────────────────────────────

Example:
───────────────────────────────────────────────────────────────
GEMINI_API_KEY=AIzaSyDXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
───────────────────────────────────────────────────────────────

IMPORTANT:
- File name must be exactly: .env (not .env.txt or env.txt)
- No spaces around the = sign
- No quotes around the API key
- Replace 'your_actual_api_key_here' with your actual key


STEP 4: RESTART THE SERVER
───────────────────────────────────────────────────────────────
Kill current server:
pkill -f "uvicorn main:app"

Start server again:
cd backend
source venv/bin/activate
uvicorn main:app --reload --host 0.0.0.0 --port 8000


STEP 5: VERIFY IT'S WORKING
───────────────────────────────────────────────────────────────
Check backend terminal. You should see:
✅ [AUDIT] Loaded API key from .env file
✅ [AUDIT] Gemini API configured successfully

If you see:
⚠️ [AUDIT WARNING] GEMINI_API_KEY not found
Then check:
- Is .env file in backend/ directory?
- Is API key correct?
- No spaces or quotes?
